## Jinja2

### 템플릿 엔진이란?

- 정적인 HTML에 동적인 데이터를 결합하여 웹 페이지를 생성합니다.

- 즉, 템플릿 파일에 데이터를 삽입하여 동적인 HTML을 생성하는 것입니다.

### Jinja2

- Python에서 가장 많이 사용되는 템플릿 엔진입니다.

- 주로 Flask에서 채택되어 사용되고 있습니다.

#### 기본 예시

```
from jinja2 import Template

# 템플릿

# 데이터

# 렌더링 : 템플릿에 데이터를 삽입하여 최종 HTML 생성
```

**참고**

- 변수 사용 : `{{ }}`

- 주석 : `{# #}`

- 조건문 : `{% if ... %}` , `{% else %}` ,  `{% endif %}`

- 반복문 : `{for ... %}` ... `{% endfor %}`
